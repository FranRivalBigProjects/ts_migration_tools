What the Analyzer Measures

The analyzer scans your code across six dimensions to produce a migration health score and risk assessment.

  • Type safety — any usage, missing annotations, implicit returns
  • Declaration quality — var count, reassignment patterns, scoping issues
  • Code complexity — function count, nesting depth, file size
  • API patterns — CommonJS require, fetch/HTTP calls, class structure
  • Equality safety — loose == / != operators throughout the code
  • ES version indicators — older patterns that conflict with strict TS
