What the Refactor Assistant Covers

Choose a focus mode or run a full refactor. Each plan is generated based on patterns detected in your actual code.

  • Type Safety — replace any, annotate parameters, extract interfaces, add generics
  • Modernize Syntax — var → const/let, loose equality, template literals, arrow functions
  • Async Patterns — callback → Promise → async/await, typed Promise return types
  • Structure & Classes — prototype → class, module extraction, single responsibility
  • Full Refactor — all of the above, prioritized by impact and safety
